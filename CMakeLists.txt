############### KDEPIM-Runtime version ################
# KDEPIM_RUNTIME_VERSION
# Version scheme:KDEPIM_RUNTIME_VERSION_NUMBER "5.7.40".
#
# x is the version number.
# y is the major release number.
# z is the minor release number.
#
#KDEPIM_RUNTIME_VERSION_NUMBER "5.7.40" follow the kdelibs version kdepim is released with.
#
# IfKDEPIM_RUNTIME_VERSION_NUMBER "5.7.40"
#
# KDEPIM_RUNTIME_DEV_VERSION
# is empty for final versions. For development versionsKDEPIM_RUNTIME_VERSION_NUMBER "5.7.40" is
# something likeKDEPIM_RUNTIME_VERSION_NUMBER "5.7.40".
#
# Examples in chronological order:
#
#    3.0
#    3.0.1
#    3.1 
#    3.1 beta1
#    3.1 beta2
#    3.1 rc1
#    3.1
#    3.1.1
#    3.2 pre
#    3.2
